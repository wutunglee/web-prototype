先從訓練狀態開始
再選擇補給策略

PulsarPump 不鼓勵「看成分就買」。 我們把補給視為訓練的一部分， 先理解你此刻的訓練狀態，再決定該用哪一套策略。

以下四種情境，是多數運動者最常遇到的狀態。 你不需要一次選對全部，只要先找到最貼近現在的那一種。

你現在最接近哪一種訓練狀態？

🔥 炎熱高溫

在高溫下維持穩定輸出

適合悶熱、流汗量大或容易補給失序的情境。 重點不是補更多，而是讓能量與電解質輸出保持穩定。

查看建議系統 → 穩定輸出（Performance Core）
🏃 長距離耐力

把節奏撐到後段

適合長時間訓練、馬拉松或耐力型活動。 這條路線處理的是時間累積後的節奏與補給穩定性。

若你在高訓練量週期中，感覺「課表越來越難完成」， 可能需要進一步處理能量重建。

查看建議系統 → 穩定輸出（Performance Core）
延伸判斷 → 能量重建（Energy Rebuild）
⚡ 高強度啟動

比賽日、關鍵課表與接近力竭的時刻

適合高強度訓練、間歇、重量訓練或比賽日。 不只幫助你進入工作狀態， 也在接近力竭時，協助你把輸出再往前推一點。

查看建議系統 → 兩段式高強度啟動
🔄 恢復與重建

讓身體回到可訓練狀態

適合高訓練量週期、作息不穩或需要長期累積的狀態。 這條路線處理的是日常恢復底盤， 而不是能量再生或輸出能力提升。

查看建議系統 → 恢復與重建（Recovery Base）

快速對照

如果你不確定該從哪一個開始，可以用下面這些情境快速比對。

今天很熱、流汗多 → 穩定輸出
今天要跑很久 → 穩定輸出
連續多天訓練、覺得回不來 → 能量重建
今天是比賽日／最後一組 → 高強度啟動
最近訓練量很高、作息不穩 → 恢復與重建
